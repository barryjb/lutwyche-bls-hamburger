Instructions for new website

Update "Location" to actual location.
Update "Service" to actual service.
Download four 550px square images that demonstrate the four main features of the service and save into img folder in the format "service-feature-img-1".
Select suitable 3 colour scheme that makes website look good with the images selected.
Select suitable font using google fonts or similar.
